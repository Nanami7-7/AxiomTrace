# 通信协议 v1

## 1. 物理层

- UART0，PA10 TX / PA11 RX；115200 baud，8 data bits，no parity，1 stop bit。
- 命令以ASCII发送，以`\r\n`结尾；固件同时接受CR或LF结束。
- 文本遥测为UTF-8/ASCII兼容文本。JustFloat保留给VOFA+，0.1 GUI默认不启用二进制模式。

## 2. 机器响应

机器响应始终以`@`开头，字段为逗号分隔的`key=value`。

| 请求 | 响应 | 说明 |
|---|---|---|
| `Info?` | `@INFO,fw=0.1.0,proto=1,...` | 固件、协议、板卡、驱动、波特率 |
| `Config?` | `@CONFIG,ppr=13,...` | 机械参数、死区、遥测周期与通道数 |
| `Status?` | `@STATUS,motor=当前,...` | 当前选中电机状态 |
| `Status=0..3` | `@STATUS,motor=n,...` | 指定电机状态 |
| `Stream=1` | `@ACK,cmd=stream,state=1` | 进入遥测循环，不使能电机 |
| `Stream=0` | `@ACK,cmd=stream,state=0` | 退出遥测循环，不改变电机状态 |

## 3. 电机与调参命令

| 命令 | 参数范围 | 作用/注意 |
|---|---:|---|
| `Motor=n` | n=0..3 | 选择后续调参对象；0=A/M1/RB，1=B/M2/RF，2=C/M3/LF，3=D/M4/LB |
| `Kp=x`/`Ki=x`/`Kd=x` | -100..100 | 设置当前电机速度环PID |
| `Target=x` | -800..800 RPM | 设置当前电机目标速度 |
| `Run` / `Run=n` | n=0..3 | 复位PID并使能电机；兼容行为：进入遥测循环 |
| `Stop` / `Stop=n` | n=0..3 | 停止电机并退出遥测循环 |
| `StopAll` | 无 | 停止四路电机并退出遥测循环 |
| `FFk=x` / `FFb=x` | 有限浮点数 | 前馈线性参数 |
| `FFe=0/1` | 0或非0 | 禁用/使能前馈 |
| `FFKp/Ki/Kd=x` | -100..100 | 前馈模式PID修正参数 |

## 4. 运动控制命令

| 命令 | 参数 | 说明 |
|---|---|---|
| `mode=speed` | 无 | 速度控制 |
| `mode=position` | 无 | 位置控制 |
| `mode=angle` | 无 | 角度控制 |
| `pos=pulses,cruiseRPM` | cruiseRPM 0..规划上限 | 四轮平均编码器位移 |
| `angle=degree,cruiseRPM` | cruiseRPM 0..规划上限 | 依据IMU yaw转向 |
| `abort` | 无 | 紧急终止位置控制并停止全部电机 |

`Step`、`Auto`、`Sweep`为实验室辨识命令，可能持续数秒至数十秒；0.1 GUI不提供快捷按钮，必须在车辆可靠支撑、限流供电条件下从终端手动执行。

## 5. FireWater遥测

周期：30 ms；每行11个十进制浮点数，以逗号分隔并以LF结束。

| CH | 名称 | 单位/含义 |
|---:|---|---|
| 0..3 | rpm_a..rpm_d | 四路实测RPM |
| 4..7 | target_a..target_d | 四路目标RPM |
| 8 | selected_ff | 当前选中电机前馈输出 |
| 9 | selected_pid_correction | 当前选中电机PID修正量 |
| 10 | selected_output | 当前选中电机最终有符号驱动命令（-500..500） |

示例：

```text
0.000000,0.000000,0.000000,0.000000,100.000000,0.000000,0.000000,0.000000,0.000000,12.300000,112.000000
```

## 6. JustFloat

`float32 little-endian × N + 00 00 80 7F`。不得在同一串口会话中无状态地混用二进制JustFloat和文本命令；GUI 0.1只解析FireWater。

## 7. 安全顺序

1. 连接后查询`Info?`，确认`proto=1`。
2. 发送参数并用`Status=n`读回。
3. 首次运行悬空车轮，设置低目标RPM。
4. `Run=n`启动；需要绘图但不启动电机时用`Stream=1`。
5. 断开串口前发送`StopAll`，再发送`Stream=0`。
